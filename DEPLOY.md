# Deploying to Vercel

## Quick Deploy (Recommended)

1. **Run this command in your terminal:**
   ```bash
   cd "/Users/macbookair/Desktop/portfolio 2"
   npx vercel
   ```

2. **Follow the prompts:**
   - Login to Vercel (you'll be redirected to browser)
   - Press Enter to confirm project settings
   - Press Enter to deploy to production

## Alternative: Deploy via Web Interface

1. Go to [vercel.com](https://vercel.com) and sign up/login
2. Click "Add New Project"
3. Import your Git repository (if you push to GitHub) OR
4. Use "Deploy" and drag-drop your project folder

## After Deployment

- Your site will be live at: `https://your-project-name.vercel.app`
- Vercel provides automatic HTTPS
- Every git push can trigger automatic deployments (if connected to GitHub)
